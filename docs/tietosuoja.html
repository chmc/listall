ListAllin tietosuojakäytäntö

Voimaantulopäivä: 25. lokakuuta 2025
Viimeksi päivitetty: 25. lokakuuta 2025

Yleiskatsaus

ListAll ("me", "meidän" tai "sovellus") on sitoutunut suojaamaan yksityisyyttäsi. Tämä tietosuojakäytäntö selittää, miten ListAll käsittelee tietojasi, kun käytät iOS- ja watchOS-sovelluksiamme.

Tietosuojasitoumuksemme

ListAll on suunniteltu yksityisyys ydinperiaatteena:

  • EMME kerää henkilötietoja
  • EMME seuraa käyttöäsi tai käyttäytymistäsi
  • EMME käytä analytiikka- tai seurantapalveluita
  • EMME jaa tietojasi kolmansille osapuolille
  • EMME näytä mainoksia
  • EMME vaadi tilin luomista tai kirjautumista

Tietojen tallennus ja kerääminen

Mitä tietoja tallennetaan paikallisesti

ListAll tallentaa seuraavat tiedot vain paikallisesti laitteellesi:

  1. Listat: Listojesi nimet ja järjestys
  2. Kohteet: Kohteiden otsikot, kuvaukset, määrät ja valmistumistila
  3. Kuvat: Kohteisiin liittämäsi kuvat (tallennetaan sovelluksen paikalliseen tietokantaan)
  4. Käyttäjäasetukset: Sovellusasetukset kuten Face ID -lukitusasetus, teemavalinta
  5. Ehdotushistoria: Paikallinen historia luomistasi kohteista (käytetään älykkäisiin ehdotuksiin)

Kaikki nämä tiedot tallennetaan:

  • Laitteesi suojattuun tallennustilaan (Core Data -tietokanta)
  • App Groups -säilöön (iPhonen ja Apple Watchin väliseen synkronointiin)

iCloud-synkronointi

ListAll käyttää iCloudia saumattomaan synkronointiin laitteidesi välillä:

  • Listatietosi synkronoidaan automaattisesti henkilökohtaiselle iCloud-tilillesi
  • Tiedot salataan siirron aikana ja levossa Applen iCloud-tietoturvalla
  • MEILLÄ EI ole pääsyä iCloud-tietoihisi
  • Vain sinä voit käyttää synkronoituja tietojasi Apple ID:lläsi
  • Toimii automaattisesti kun olet kirjautunut iCloudiin laitteillasi

Tiedot joita EMME kerää

ListAll EI kerää, lähetä tai tallenna:

  • Henkilökohtaisia tunnistetietoja (nimi, sähköposti, puhelinnumero)
  • Sijaintitietoja tai GPS-koordinaatteja
  • Käyttöanalytiikkaa tai sovelluksen käyttäytymistietoja
  • Laitetunnisteita tai mainostunnisteita
  • Selaushistoriaa tai hakukyselyitä
  • Yhteystietoja tai osoitekirjan tietoja
  • Mitään tietoja laitteesi muista sovelluksista

Pyytämämme käyttöoikeudet

ListAll pyytää seuraavia käyttöoikeuksia vain tarvittaessa:

Kameran käyttöoikeus

  • Miksi: Kuvien ottamiseen kohteisiin liitettäväksi
  • Milloin: Vain kun napautat "Ota kuva" -painiketta
  • Tietojen tallennus: Kuvat tallennetaan vain paikallisesti sovelluksen tietokantaan

Kuvakirjaston käyttöoikeus

  • Miksi: Olemassa olevien kuvien valitsemiseen kohteisiin liitettäväksi
  • Milloin: Vain kun napautat "Valitse kirjastosta" -painiketta
  • Pääsy: Rajoitettu vain kuviin, jotka nimenomaisesti valitset

Face ID / Touch ID -käyttöoikeus

  • Miksi: Valinnainen sovelluksen lukitusominaisuus turvallisuutta varten
  • Milloin: Vain jos otat käyttöön "Lukitse sovellus" Asetuksissa
  • Tiedot: Biometriset tiedot eivät koskaan poistu laitteeltasi (iOS käsittelee)

iCloud-käyttöoikeus (valinnainen)

  • Miksi: Listojesi synkronoimiseen Apple-laitteidesi välillä
  • Milloin: Vain jos nimenomaisesti otat käyttöön iCloud-synkronoinnin Asetuksissa
  • Pääsy: Vain henkilökohtainen iCloud-tilisi

Tietojen jakaminen

ListAll EI jaa tietojasi:

  • Kolmannen osapuolen palveluille tai analytiikkayhtiöille
  • Mainosverkostoille
  • Sosiaalisen median alustoille
  • Muille sovelluksille tai palveluille
  • Sovelluksen kehittäjälle (meille)

Kun valitset jakaa:

ListAll sisältää vienti/jako-ominaisuuden, joka mahdollistaa listojen jakamisen muiden kanssa. Tämä jakaminen on:

  • Täysin vapaaehtoista ja käyttäjän aloittamaa
  • Käyttää iOS:n sisäänrakennettua jakolevyä (sinun valitsemasi menetelmä)
  • Ei sisällä meidän palvelimiamme tai kolmannen osapuolen palveluita

Tietoturva

Tietojesi turvallisuutta suojaa:

  • Paikallinen tallennus: Kaikki tiedot tallennetaan iOS:n turvalliseen Core Data -tietokantaan
  • Sovelluksen eristys: iOS:n sovelluksen eristys estää muita sovelluksia pääsemästä ListAllin tietoihin
  • Valinnainen biometrinen lukitus: Face ID/Touch ID -suojaus saatavilla
  • iCloud-salaus: Jos synkronointi käytössä, tiedot salataan Applen iCloud-infrastruktuurilla

Lasten yksityisyys

ListAll ei tietoisesti kerää henkilötietoja keneltäkään, mukaan lukien alle 13-vuotiailta lapsilta. Koska emme kerää mitään tietoja, sovellus on turvallinen kaiken ikäisille käyttäjille. Sovelluksen ikäraja on 4+ (Ei sopimattomalta sisältöä) App Storessa.

Oikeutesi ja hallinta

Sinulla on täysi hallinta tietoihisi:

  • Pääsy: Kaikki tietosi on tallennettu paikallisesti ja käytettävissä sovelluksen kautta
  • Vienti: Vie kaikki tietosi JSON-, CSV- tai pelkkä teksti -muodossa
  • Poisto: Poista yksittäisiä listoja/kohteita tai kaikki tiedot sovelluksen kautta
  • Synkronoinnin poisto käytöstä: Poista iCloud-synkronointi käytöstä milloin tahansa Asetuksissa
  • Asennuksen poisto: Sovelluksen poistaminen poistaa kaikki paikalliset tiedot laitteeltasi

Kolmannen osapuolen palvelut

Ei mitään. ListAll ei käytä mitään kolmannen osapuolen palveluita, SDK:ita tai kirjastoja, jotka keräävät tietoja.

Muutokset tähän tietosuojakäytäntöön

Saatamme päivittää tätä tietosuojakäytäntöä ajoittain. Ilmoitamme muutoksista:

  • Julkaisemalla uuden tietosuojakäytännön sovelluksessa ja tällä verkkosivustolla
  • Päivittämällä "Viimeksi päivitetty" -päivämäärän tämän käytännön yläosassa
  • Merkittävistä muutoksista näyttämällä ilmoituksen sovelluksessa

Yhteenveto

ListAll on suunniteltu täysin yksityiseksi. Tietosi eivät koskaan poistu laitteiltasi (ellet nimenomaisesti ota käyttöön iCloud-synkronointia, joka käyttää henkilökohtaista iCloud-tiliäsi). Emme kerää, seuraa tai jaa mitään. Listasi ovat vain sinun.
